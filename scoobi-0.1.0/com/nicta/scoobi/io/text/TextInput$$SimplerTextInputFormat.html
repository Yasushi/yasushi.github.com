SimplerTextInputFormat

class SimplerTextInputFormat extends FileInputFormat[NullWritable, StringWritable]

A wrapper around TextInputFormat that changes the type paramerterisation from LongWritable-Text to NullWritable-StringWritable. This input format is then used as the basis for loading files into a Scoobi job.
